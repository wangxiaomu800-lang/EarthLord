## 1. 游戏概念

### 1.1 游戏概述
地球新主是一款结合真实地理位置的末日生存开拓游戏。玩家在现实世界中行走,通过GPS圈占虚拟领地,探索真实地标获取资源,建造家园,与其他玩家交易和社交,在废土世界中重建文明。

### 1.2 核心理念
**"用脚步丈量你的王国"**

- **Walk to Earn**: 真实步行转化为游戏进度
- **希望朋克**: 废土背景下的新生与希望
- **真实映射**: 虚拟游戏与真实世界深度绑定
- **社交互动**: 基于地理位置的全新社交体验

### 1.3 游戏特色
1. **独特的圈地玩法**: 像小时候在沙滩画圈一样,用GPS圈占真实土地
2. **真实POI探索**: 医院、超市、公园等真实地标变成游戏资源点
3. **本地社交**: 只有附近的人才能交易和聊天,强地理属性
4. **生存压力**: 饥渴系统制造紧迫感,驱动玩家持续游玩
5. **希望主题**: 正能量的末日题材,重建而非毁灭

## 2. 故事背景

### 2.1 世界观

#### 时间线
**2048年 - "终末战争"爆发**
- 全球文明在72小时内崩塌
- 99%的人口消失
- 城市变成废墟,秩序不复存在

**2051年 - 新世界的诞生**
- 幸存者从避难所走出
- 《新开拓者协议》颁布
- 新的规则开始运行

#### 核心法则
> **《新开拓者协议》第一条:**
> "凡以双足丈量之无主土地,皆为开拓者所有。"

这条法则改变了一切:
- 旧世界的产权失效
- 用脚步丈量即可占有土地
- GPS卫星系统成为仲裁者
- 步行成为最有价值的行为

### 2.2 游戏开场

#### 序章剧情
玩家从避难所醒来,看到门口终端机的提示:

```
>>> 终端机 - 系统信息 <<<

距离"终末战争"已过去1095天

《新开拓者协议》现已生效

请登记身份,开始你的开拓之旅

> 登录系统
> 查看协议
> 退出 (不可用)
```

玩家完成登录后,收到第一条消息:

```
[官方广播]
欢迎加入新世界,开拓者。

旧世界已经不存在了。
但希望,从未消失。

走出去,用你的双脚丈量这片土地。
这里的每一寸,都可能成为你的王国。

重建文明的使命,交给你了。

——《新开拓者协议》管理委员会
```

### 2.3 视觉风格: 希望朋克

**不是传统的黑暗末日,而是充满希望的废土重生**

**环境设计**:
- 破败建筑上爬满绿色植物
- 废弃工厂被改造成温室
- 荒芜的街道种满庄稼
- 卫星信号塔在废墟上闪烁

**色彩方案**:
- 基调:灰色废墟(旧世界)
- 点缀:绿色植物、蓝色能源、金色光芒(新世界)
- 对比:绝望与希望的交织

**UI风格**:
- 末日感:做旧的金属质感,全息投影效果
- 科技感:卫星地图,GPS轨迹线
- 温暖感:圆润边角,柔和光效

#### 玩家动机
1. **收集欲**: 探索地标,收集资源
2. **建造欲**: 建设家园,解锁建筑
3. **竞争欲**: 排行榜,领地面积比拼
4. **社交欲**: 和附近玩家交易聊天
5. **成就感**: 解锁成就,等级提升

### 3.2 五大核心系统

#### 系统1: 圈地系统
**设计理念**: 让步行变得有意义

**玩法流程**:
1. 打开App,点击"开始圈地"
2. 沿着想要占领的区域边界行走
3. GPS记录轨迹,绘制蓝色线条
4. 走回起点附近,轨迹自动闭合
5. 系统验证(不重叠、不作弊)
6. 验证通过,领地永久归属玩家

**设计细节**:
- **视觉反馈**: 轨迹线由蓝色变为绿色,配合震动反馈
- **即时奖励**: 显示面积和奖励金币
- **仪式感**: 播放"领地确认"动画和音效

**平衡性设计**:
- 最小面积: 100m² (防止垃圾圈地)
- 最大面积: 100,000m² (防止垄断)
- 冷却时间: 每次圈地间隔30分钟
- 活跃要求: 30天内无建造会警告,90天回收

**防作弊机制**:
- 速度检测: 超过15km/h判定为交通工具,无效
- 轨迹合理性: 检测瞬移、直线等异常
- 设备检测: 识别模拟器和虚拟定位

#### 系统2: 探索系统
**设计理念**: 把通勤和运动变成游戏

**玩法流程**:
1. 玩家移动到POI(兴趣点)100米范围内
2. 屏幕弹出提示:"发现 [XX医院]"
3. 点击"搜刮"按钮
4. 播放探索动画(3秒)
5. 展示获得的资源和探索成就

**POI与资源映射**:
| POI类型 | 主要产出 | 稀有产出 | 掉落率 |
|---------|----------|----------|--------|
| 医院/药店 | 急救包×5-10 | 抗生素×1-3 | 普通90%,稀有10% |
| 超市/餐厅 | 罐头×10-20,水×10-20 | 高级食材×1-5 | 普通95%,稀有5% |
| 工厂/工地 | 金属×10-15,螺丝×20-30 | 精密零件×1-3 | 普通90%,稀有10% |
| 公园/森林 | 木材×15-25,纤维×10-15 | 稀有植物×1-2 | 普通95%,稀有5% |
| 银行/商场 | 布料×5-10,玻璃×5-10 | 卫星模块×1 | 普通80%,稀有20% |

**奖励计算公式**:
```
基础奖励 = POI类型基础值
距离加成 = 探索距离(km) × 10%
稀有度加成 = 随机(0-100),大于阈值则掉落稀有物品

最终奖励 = 基础奖励 × (1 + 距离加成) × 稀有度加成
```

**冷却机制**:
- 每个POI每24小时可搜刮1次
- VIP用户冷却时间减半(12小时)
- 特殊活动期间冷却时间减半

**成就系统整合**:
- 探索1个POI → 末世拾荒者(50金币)
- 探索10个POI → 城市探险家(100金币)
- 探索100个POI → 废墟专家(1000金币)
- 探索500个POI → 传奇探索者(10000金币)

#### 系统3: 建造系统
**设计理念**: 资源消耗出口,长期目标驱动

**建筑分级体系**:

**Tier 1 - 生存基础** (游戏前期,1-7天)
| 建筑名称 | 所需资源 | 建造时间 | 功能 |
|----------|----------|----------|------|
| 篝火 | 石头×20,木材×30 | 10分钟 | 提供温暖,附近玩家可见 |
| 简易庇护所 | 木材×50,布料×20 | 30分钟 | 恢复生命值 |
| 小仓库 | 木材×40,金属×10 | 20分钟 | 背包容量+20 |
| 水井 | 石头×30,金属×15 | 40分钟 | 每小时产出水×5 |

**Tier 2 - 功能扩展** (游戏中期,7-30天)
| 建筑名称 | 所需资源 | 建造时间 | 功能 |
|----------|----------|----------|------|
| 工作台 | 木材×60,金属×30 | 1小时 | 可制作工具 |
| 农田 | 木材×20,种子×5 | 2小时 | 每小时产出食物×3 |
| 中型仓库 | 木材×80,金属×40 | 1.5小时 | 背包容量+50 |
| 营地电台 | 金属×100,电子元件×10 | 2小时 | 通讯范围扩大到10km |

**Tier 3 - 高级设施** (游戏后期,30天+)
| 建筑名称 | 所需资源 | 建造时间 | 功能 |
|----------|----------|----------|------|
| 太阳能板 | 金属×100,电子元件×20 | 3小时 | 产出能源,供给其他建筑 |
| 风力发电机 | 金属×150,电子元件×30 | 4小时 | 产出更多能源 |
| 大型仓库 | 木材×200,金属×100 | 4小时 | 背包容量+100 |
| 卫星通讯站 | 稀有材料×50,卫星模块×5 | 6小时 | 全球通讯 |
| 防御塔 | 金属×200,电子元件×50 | 5小时 | 防止领地被侵占(PVP) |

**建筑状态机**:
```
[待建造] → 点击建造 → [建造中] → 倒计时结束 → [待激活]
                                                    ↓
                                                点击激活
                                                    ↓
                                                [运行中] ← 定期维护
                                                    ↓
                                           不维护或受损
                                                    ↓
                                                [损坏] → 消耗资源修复 → [运行中]
                                                    ↓
                                               长期不修复
                                                    ↓
                                                [废弃]
```

**建筑放置规则**:
- 只能在自己的领地上建造
- 建筑间距≥10米
- 每块领地最多建造20个建筑
- 建筑可拆除,返还50%资源

**建筑升级系统**:
- 每个建筑可升级到3级
- 升级消耗资源递增(2级消耗×2,3级消耗×3)
- 升级效果提升50%
- 例:小仓库1级+20格,2级+40格,3级+60格

**设计理念**: 创造玩家互动,资源流通

**交易机制**:

4. 玩家B看到挂单列表
5. 玩家B接受交易

**交易触发条件**:
- 双方必须在100米范围内
- 领地主设置"允许交易"

**交易经济平衡**:

**资源稀有度分级**:
- **普通**: 食物、水、木材、石头
- **稀有**: 金属、药品、电子元件
- **史诗**: 卫星模块、高级材料

#### 系统5: 生存系统
**设计理念**: 创造紧迫感,驱动持续游玩

**三大体征**:

**1. 核心生命值**
- 初始值: 100
- 最大值: 200 (通过升级提升)
- 降到0 → 角色死亡
- 死亡惩罚: 复活后生命值50%,背包掉落10%物品

**2. 饱食度**
- 初始值: 100%
- 每小时衰减: -5%
- 低于20% → 生命值每小时-10
- 恢复方式: 食用罐头(+30%),高级食材(+50%)

**3. 水分**
- 初始值: 100%
- 每小时衰减: -10% (比饥饿快)
- 低于20% → 生命值每小时-15
- 恢复方式: 饮用水(+40%),纯净水(+60%)

**增益机制**:
- 饱食度>80% 且 水分>80% → 探索奖励+20%
- 饱食度>80% 且 水分>80% → 建造速度+10%
- 全满状态 → 移动速度+5%

**时间衰减计算**:
```
每10分钟检查一次:

应扣除饱食度 = (当前时间 - 上次检查时间) / 1小时 × 5%
应扣除水分 = (当前时间 - 上次检查时间) / 1小时 × 10%

if 饱食度 < 20%:
    应扣除生命值 += (当前时间 - 上次检查时间) / 1小时 × 10

if 水分 < 20%:
    应扣除生命值 += (当前时间 - 上次检查时间) / 1小时 × 15
```

**平衡性设计**:
- 普通玩家: 每24小时需要食物×2,水×3
- VIP玩家: 衰减速度-30%
- 被动产出: 建造"农田"和"水井"可自动生产

## 4. 通讯与社交系统

### 4.1 通讯设备分级

| 设备名称 | 解锁条件 | 覆盖范围 | 功能 | 价格 |
|----------|----------|----------|------|------|
| 收音机 | 初始拥有 | 3km(只读) | 只能接收公共频道消息 | 免费 |
| 对讲机 | 等级5 | 3km(读写) | 可发送消息到公共频道 | $2.99或500金币 |
| 营地电台 | 建造 | 10km(读写) | 扩大通讯范围,创建频道 | 建造材料 |
| 卫星通讯 | VIP或建造 | 全球(读写) | 全球聊天,私聊 | $9.99/月或建造 |

### 4.2 公共频道

**设计理念**: 模拟真实无线电,基于地理位置的匿名社交

**频道规则**:
- 3km范围内的玩家自动加入同一公共频道
- 玩家移动 → 频道自动切换
- 消息保留24小时后自动删除
- 显示发送者昵称+距离(如:"玩家ABC · 1.2km")

**反垃圾机制**:

### 4.3 频道管理

**自定义频道**:
- 任何玩家可创建频道
- 频道类型: 公开频道、私人频道
- 频道名称: 最多20字符
- 频道描述: 最多100字符

**频道发现**:
- 展示附近10km内的所有公开频道
- 显示频道名称、在线人数、距离
- 支持搜索和排序

**频道管理**:
- 频道创建者 = 管理员
- 可踢出成员、删除消息
- 可设置频道公开/私密
- 可转让管理员权限

## 5. 进阶系统

### 5.1 等级系统

**经验来源**:
| 行为 | 经验值 |
|------|--------|
| 圈地1次 | 50 XP |
| 探索1个POI | 20 XP |
| 建造1个建筑 | 30-100 XP |
| 完成1次交易 | 10 XP |
| 在线1小时 | 5 XP |

**等级曲线**:
```
Level 1 → 2: 100 XP
Level 2 → 3: 200 XP
Level 3 → 4: 400 XP
```

## 10. 付费点设计

## 11. 反作弊系统

### 11.1 GPS作弊检测

**虚拟定位检测**:
- iOS原生检测: `location.sourceInformation`
- 速度检测: 超过15km/h标记为"交通工具移动"
- 轨迹合理性: 检测瞬移(两点间距离/时间>人类步速)

**处罚机制**:
- 第1次: 警告,本次圈地无效
- 第2次: 禁止圈地24小时
- 第3次: 永久封禁

### 11.2 刷资源检测

**小号倒资源**:
- 检测异常交易: A给B稀有资源,B给A垃圾
- 账号关联: 相同设备/IP/地理位置
- 处罚: 回滚交易,封禁账号

**脚本挂机**:
- 行为模式分析: 完全相同的操作间隔
- 验证码: 连续探索10次后弹出验证
- 处罚: 封禁账号
